# 小红书 30 天运营包（Agent 学习 IP）

## 文件说明
- `00-小红书30天运营策略与执行方案.md`：完整策略与执行方案
- `01-首周发布脚本模板.md`：首周 3 视频 + 2 图文可直接改稿模板
- `02-数据看板与复盘模板.md`：数据结构、周复盘与结案模板
- `03-今日发布包-2026-02-10.md`：今天可直接发布的视频脚本、文案、评论话术
- `04-本周作战卡-2026W1.md`：本周排期、验收标准与风险提醒
- `templates/content-calendar-30d.csv`：30 天排期样表（20 条）
- `templates/performance-snapshot-template.csv`：内容表现快照模板
- `templates/weekly-review-template.md`：周复盘空白模板

## 推荐使用顺序
1. 先读 `00-...` 锁定定位、节奏和 KPI。
2. 用 `templates/content-calendar-30d.csv` 确认排期。
3. 用 `01-...` 直接产出首周内容。
4. 今天优先执行 `03-今日发布包-2026-02-10.md`。
5. 每周用 `weekly-review-template.md` 做复盘迭代。
